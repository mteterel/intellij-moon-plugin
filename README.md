# moonrepo IntelliJ plugin (unofficial)

![Build](https://github.com/mteterel/intellij-moon-plugin/workflows/Build/badge.svg)
[![Version](https://img.shields.io/jetbrains/plugin/v/32442.svg)](https://plugins.jetbrains.com/plugin/32442)
[![Downloads](https://img.shields.io/jetbrains/plugin/d/32442.svg)](https://plugins.jetbrains.com/plugin/32442)

moon console (unofficial) is an IntelliJ plugin that acts as a control panel for Moonrepo.

It surfaces Moonrepo projects, tasks, templates, and maintenance actions inside the IDE so you can stay in your editor while working with a monorepo.

## Installation

- Using the IDE built-in plugin system:

  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Marketplace</kbd> > <kbd>Search for "moon console (unofficial)"</kbd> >
  <kbd>Install</kbd>

- Using JetBrains Marketplace:

  Go to [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/32442) and install it by clicking the <kbd>Install to ...</kbd> button in case your IDE is running.

  You can also download the [latest release](https://plugins.jetbrains.com/plugin/32442/versions) from JetBrains Marketplace and install it manually using
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from disk...</kbd>

- Manually:

  Download the [latest release](https://github.com/mteterel/intellij-moon-plugin/releases/latest) and install it manually using
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from disk...</kbd>
